Dua tombol: "YA, LANJUTKAN" (hijau) dan "TIDAK, BATALKAN" (merah)
Pesan jelas: "Skrining tidak direkomendasikan untuk pasien perempuan"
Jika klik "TIDAK" → script berhenti, tidak ada pengisian form
Jika klik "YA" → lanjutkan dengan gender="1"
